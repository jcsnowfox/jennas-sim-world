# Privacy Boundaries

## Local by default

The following remain local by default: raw films and videos, subtitles, transcripts, sampled frames, media-analysis caches, local voice caches, unapproved memory candidates, private city saves, and local diagnostics unless explicitly exported.

## Ghostlight-enabled synchronisation

When the user enables Ghostlight and selects an appropriate sync scope, the product may synchronise approved memories, companion continuity events, conversations, shared rituals, watch-session summaries, approved companion reflections, and selected world state where configured.

## Hard media boundary

Raw local media must never synchronise by default. Full transcripts, frame caches, and detailed scene analysis remain local unless a later explicit feature is designed, documented, consented to, and tested.
